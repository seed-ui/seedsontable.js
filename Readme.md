## Install

npm:
```
npm install seedsontable
```

## Usage

node.js:
```javascript
var seedsontable = require('seedsontable');
var Seedsontable = seedsontable.Seedsontable;
var SeedsonData = seedsontable.SeedsonData;
```

browser:
```html
<script src="seedsontable.js"></script>
var Seedsontable = seedsontable.Seedsontable;
var SeedsonData = seedsontable.SeedsonData;
```
